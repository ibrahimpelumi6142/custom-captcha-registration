# Custom CAPTCHA For Wordpress Custom Login & Registration Page

This WordPress plugin customizes Wordfence CAPTCHA validation for custom login & registration pages to reduce spam registrations and fix the "Registration Attempt Blocked" issue on custom pages.  

## Features
- Adjusts CAPTCHA display on registration page.
- Fixes the "Registration Attempt Blocked" issue on custom login & registration pages.
- Detects bots via user agent.
- Checks for spammy query parameters.

## Installation
1. Upload the `custom-captcha-registration` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress admin panel.
3. The plugin will automatically adjust CAPTCHA on the login & registration page.
